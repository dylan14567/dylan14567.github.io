SERVIDOR WEB | TERMUX

2 Abril del 2020

Este servidor web es de HTML para termux, ademas es privado, porque este servidor es un localhost y este es para utilizarlo en proyectos html, como ejemplo, crear tu propia página web, para acceder al localhost en web escribe en Google 127.0.0.1:8080.

Guia de instalación en Termux git clone https://github.com/dylan14567/Servidor-WEB-Termux Mira este video para la guia completa: https://youtu.be/w7UFd14P7pU
