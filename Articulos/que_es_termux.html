Que es Termux

Termux es un emulador de terminal Android y una aplicación de entorno Linux que funciona directamente sin necesidad de enrutamiento o configuración. Un sistema base mínimo se instala automáticamente, hay paquetes adicionales disponibles utilizando el administrador de paquetes APT .

Como tener letras verdes infinitas en la Terminal Termux:

Para eso nesecitas ir a:

cd $PREFIX/etc

cd ../usr/etc;ls

Y luego instalar el editor vim con este comando.

pkg install vim -y

Y luego edita con vim esto:

vim bash.bashrc

Luego dentro de bash.bashrc, pega este comando.

PS1='\033[1;91mroot@termux[\033[1;93m\W\033[1;91m]:# \033[1;92m'

Listo ya tienes letras verdes infinitas en la Terminal Termux.
