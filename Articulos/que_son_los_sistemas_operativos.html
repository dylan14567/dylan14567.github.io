Que son los Sistemas Operativos

Un sistema operativo es el software principal o conjunto de programas de un sistema informático que gestiona los recursos de hardware y provee servicios a los programas de aplicación de software, ejecutándose en modo privilegiado respecto de los restantes.​ Wikipedia

Sistema Operativo Microsoft Windows:

Windows es el nombre de una familia de distribuciones de software para PC, teléfonos inteligentes, servidores y sistemas empotrados, desarrollados y vendidos por Microsoft y disponibles para múltiples arquitecturas, tales como x86, x86-64 y ARM. WikipediaLanzamiento inicial: 20 de noviembre de 1985 (34 años, 5 meses y 29 días)Fecha De Lanzamiento: 20 de noviembre de 1985Interfaz de usuario predeterminada: Windows ShellEscrito en: C++, C#, JavaScript, Visual BasicActualizar métodos: Windows Update, Windows Anytime Upgrade, Microsoft Store, Windows Server Update ServicesFamilia de SO: Windows CE, Windows NT

Sistema Operativo Android:

Android es un sistema operativo móvil desarrollado por Google, basado en Kernel de Linux y otros software de código abierto. Fue diseñado para dispositivos móviles con pantalla táctil, como teléfonos inteligentes, tabletas, relojes inteligentes, automóviles y televisores. Wikipedia
Lanzamiento inicial: 23 de septiembre de 2008 (11 años, 7 meses y 24 días) (info)
Fecha del lanzamiento inicial: 23 de septiembre de 2008
Actualizar método: Over-the-air programming
Interfaz de usuario predeterminada: Interfaz gráfica de usuario (Multitáctil)
Familia de SO: Unix-like
Escrito en: Java, C, C++, Extensible Markup Language, Lenguaje ensamblador, Python, Más

Sistema Operativo Mac OS:


El 24 de enero de 1984, Apple Computer Inc. introdujo la computadora personal Macintosh, con el modelo Macintosh 128K, el cual incluía el sistema operativo Mac OS, conocido en esa época como System Software.​A la Macintosh a menudo se le atribuye la popularización de las interfaces gráficas de usuario. Wikipedia
Fecha De Lanzamiento: 1984
Interfaz de usuario predeterminada: Interfaz gráfica de usuario

Sistema Operativo iOS:

iOS es un sistema operativo móvil de la multinacional Apple Inc. Originalmente desarrollado para el iPhone, después se ha usado en dispositivos como el iPod touch y el iPad. No permite la instalación de iOS en hardware de terceros. Wikipedia
Fecha del lanzamiento inicial: 29 de junio de 2007
Familia de SO: macOS
Interfaz de usuario predeterminada: Cocoa Touch
Tipo de núcleo: Núcleo híbrido (XNU)
Escrito en: C, C++, Objective-C, Swift, Lenguaje ensamblador
Actualizar métodos: iTunes, Finder

Sistemas Operativos GNU/Linux:
GNU/Linux es un sistema operativo de tipo Unix, multiplataforma, multiusuario y multitarea. El sistema es la combinación de varios proyectos, entre los cuales destacan GNU, encabezado por Richard Stallman y la Free Software Foundation; además del núcleo o kernel «Linux», encabezado por Linus Torvalds. Wikipedia
Lanzamiento inicial: 1991
Sistema de gestión de paquetes: varios
Última versión estable: 5.6.11 (info); 6 de mayo de 2020 (11 días)
Última versión en pruebas: 5.7-rc4 (info); 3 de mayo de 2020 (14 días)
Tipo de núcleo: Núcleo monolítico
Modelo de desarrollo: Software libre y código abierto


Sistema Operativo Debian GNU/Linux:

Debian GNU/Linux es un sistema operativo libre, desarrollado por miles de voluntarios de todo el mundo, que colaboran a través de Internet. Wikipedia
Fecha De Lanzamiento: 15 de septiembre de 1993
Última versión en pruebas: 11.0 (Bullseye)
Última versión estable: 10.3 (Buster) (info); 8 de febrero de 2020 (3 meses y 3 días)
Sistema de gestión de paquetes: dpkg
Actualizar método: Soporte de largo plazo
Interfaces de usuario predeterminadas: GNOME, Xfce, LXDE

Sistema Operativo Ubuntu :

Ubuntu es un sistema operativo de software libre y código abierto. Es una distribución de Linux basada en Debian. Actualmente corre en computadores de escritorio y servidores. Está orientado al usuario promedio, con un fuerte enfoque en la facilidad de uso y en mejorar la experiencia del usuario. Wikipedia
Fecha del lanzamiento inicial: 20 de octubre de 2004
Interfaz de usuario predeterminada: GNOME
Sistema de gestión de paquetes: dpkg + APT, destacando front-ends como GNOME Software
Última versión estable: 20.04 LTS (Focal Fossa) (info); 23 de abril de 2020 (17 días)
Actualizar método: Software Updater
Lenguajes de programación: Python, Java, C, C++, C#

Sistema Operativo  Kali Linux:

Kali Linux es una distribución basada en Debian GNU/Linux diseñada principalmente para la auditoría y seguridad informática en general. Fue fundada y es mantenida por Offensive Security Ltd. Wikipedia
Fecha del lanzamiento inicial: 13 de marzo de 2013
Interfaz de usuario predeterminada: Xfce
Última versión estable: 2020.2 (Rolling) (info); 12 de mayo de 2020 (3 días)
Sistema de gestión de paquetes: dpkg
Tipo de núcleo: Monolítico, Micro
Modelo de desarrollo: Código abierto

Sistema Operativo Arch Linux:

Arch Linux ​ es una distribución Linux para computadoras x86-64​, arquitecturas ARM y I686 orientada a usuarios avanzados. Se compone en su mayor parte de software libre y de código abierto ​ y apoya la participación comunitaria.​ Wikipedia
Fecha del lanzamiento inicial: 11 de marzo de 2002
Última versión estable: Instalador: 1 de mayo de 2020 (11 días); (Liberación continua)
Sistema de gestión de paquetes: pacman
Interfaz de usuario predeterminada: Interfaz de línea de comandos (Bash)
Tipo de núcleo: Monolítico (Linux)
Tipo de mercado: General


Sistema Operativo Parrot Security OS:

Parrot Security OS es una distribución de Linux basada en Debian con un enfoque en la seguridad informática. Está diseñado para pruebas de penetración, evaluación y análisis de vulnerabilidades, análisis forense de computadoras, navegación web anónima, y practicar criptografía. Es desarrollado por el Frozenbox Team. Wikipedia
Fecha del lanzamiento inicial: 10 de abril de 2013
Interfaz de usuario predeterminada: MATE
