Que es Kali Linux ??

Kali Linux es una distribución basada en Debian GNU/Linux diseñada principalmente para la auditoría y seguridad informática en general. Fue fundada y es mantenida por Offensive Security Ltd. Wikipedia
Sistema de gestión de paquetes: dpkg
Última versión estable: 2020.1 (Rolling) (info); 28 de enero de 2020 (2 meses y 23 días)
Interfaz de usuario predeterminada: Xfce
Tipo de núcleo: Monolítico, Micro
Modelo de desarrollo: Código abierto
Interfaz gráfica predeterminada: GNOME, KDE, Xfce o LXDE

Que es Kali NetHunter ??

Una de las distribuciones de hacking ético más populares es Kali Linux. Se trata de una distribución desarrollada por Offensive Security, quien también desarrolla otras herramientas como un NetHunter que nos permite probar la seguridad de nuestros equipos desde nuestro dispositivo Android.
